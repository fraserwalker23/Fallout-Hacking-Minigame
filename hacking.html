Introduction

That’s right, another homemade version of the hacking minigame from Fallout 3. Though many versions already exist1, I went and made my own anyways. Before going over the finished product, I want to walk the reader through the steps in how it was made, and explain the decision making that went into the design of the project.

The Game

The Game generates 20 words from a Dictionary and designates one as the Answer. The User has 4 Attempts to guess the Answer from the choices provided. With each guess, the Game determines if (a) the guess is correct then enter the Win state, or (b) the guess is incorrect, subtract an Attempt, and then if (i) no Attempts remaining then enter Lose state or (ii) calculate a Similarity score, display to User, and ask the User to guess again.
